49个过程背诵特点：

1. 启动过程组只有：启动项目章程 和 识别干系人
2. 收尾过程组只有：结束项目和阶段
3. 规划过程组处理整体（整合）第一行是：**制定项目章程，其他的领域第一行都是规划xx管理开头**。如：规划范围管理、规划进度管理、规划成本管理、规划质量管理等等。
4. 执行过程组中:含有关键字，**管理、实施**。如指导与管理项目工作、管理项目知识、管理质量、管理团队、管理沟通等等。**实施只有2个，一个是风险应对，一个是实施采购**。
5. 监控过程组关键字：**控制、监督**。其中**沟通，风险，干系人不能用控制**，**只能用监督**。因为这些，你只能监督，你控制不了团队言论自由，也不能百分百得控制风险，更不能控制所有得干系人。只能监督。除此之外，做为项目经理，你要控制好其他得工作。
6. 规划过程组要学会承上启下得背诵：比如范围。
   （1）规划范围管理。说白了就是你应该做好哪些事情，如何做。这个跟项目章程、项目管理计划和质量要求决定的。
   （2）你要先收集需求，目的是为了记录和管理干系人的需要和需求。
   （3）收集需求后，经过讨论或者研究后就明白了这个项目应该怎么做，确定了这个项目验收标准此时就定义了范围。
   （4）定义范围后，就要将工作的目标进行拆解，用以进行工作分配，所以此时就有了创建WBS
